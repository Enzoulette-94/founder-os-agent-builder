# Règles administratives

## Résumé

Le vault doit rester clair, traçable et prudent avec les données personnelles ou confidentielles.

## Détails

- Ne pas stocker de données sensibles inutiles.
- Distinguer les notes personnelles, les notes clients et les notes destinées à être publiques.
- Obtenir l'autorisation nécessaire avant d'ajouter des informations clients.
- Citer les sources utilisées dans les réponses de l'agent.
- Garder une trace des décisions importantes dans [[../10-business/decisions|Décisions]].
- Utiliser des noms de fichiers courts, descriptifs, en minuscules et séparés par des tirets.
- Dater les runs et documenter leurs limites.
- Ne jamais placer de secret, mot de passe ou clé d'API dans le vault.

## À retenir pour l'agent Coach

Respecter [[permissions|les permissions]], signaler les informations manquantes et éviter toute exposition inutile de données.
