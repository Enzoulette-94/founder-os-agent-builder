# Founder OS Blueprint — Web Studio OS

Ce dépôt contient le **plan (blueprint)** d'un Founder OS — un système d'agents
IA destiné à assister un fondateur solo dans la gestion quotidienne de son
activité. Il ne s'agit pas d'une implémentation du système lui-même, mais de
sa conception : rôles des agents, architecture, politique de permissions,
choix des plateformes et journal de réflexion.

Le cas d'usage retenu est **Web Studio OS**, une micro-agence de
développement web assistée par IA (client cible, offre, tâches répétitives à
déléguer aux agents — voir [docs/business-brief.md](docs/business-brief.md)).
